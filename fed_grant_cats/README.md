# Federal Grants Analysis

Classifying and analyzing environmental exposure research grants from NSF and EPA (NIH analysis is in `grant_categorization/`).

## Folders
- `epa/` - EPA grant data from USAspending.gov
- `nsf/` - NSF grant data from NSF Awards API
- `agency_comparison/` - Cross-agency comparison (NSF/EPA/NIH) for Sankey diagrams
- `outputs/` - Classification results and batch outputs
- `scripts/` - Data fetching and classification scripts
- `test_batches/` - Test/sample batches for validation

## Data Sources
- EPA: USAspending.gov API
- NSF: NSF Awards Search API
